# 哥德巴赫猜想探索工具 (Goldbach Conjecture Exploration)

本仓库提供了一套用于探索哥德巴赫猜想（每一个大于 2 的偶数都可以写成两个质数之和）的数值计算和可视化分析工具。

## 项目概述

本项目主要包含以下两个部分：
1.  **Goldbach (C# WinForms)**: 基于 .NET 8 的桌面应用程序，支持交互式地生成质数并统计偶数的哥德巴赫质数对。
2.  **频谱分析 (Python)**: 使用解析数论中的圆法（Circle Method）思路，通过指数和、FFT 卷积及数值积分来分析质数及其和的分布。

## 主要功能

### C# 桌面程序
-   **质数筛选**: 快速筛选指定范围内的质数。
-   **质数对统计**: 计算偶数的哥德巴赫质数对数量。
-   **六进制支持**: 提供**六进制 (Base 6)** 的显示与解析功能。由于大于 3 的质数均分布在 6n ± 1 的形式中，六进制在研究质数规律时具有独特的数学参考价值。
-   **性能计时**: 实时显示计算耗时。

### Python 可视化 (`plot_S_alpha.py`)
-   **频谱和计算**: 计算关于质数的指数和 S(α)。
-   **FFT 卷积**: 利用快速傅里叶变换 (FFT) 快速统计质数有序对的和分布。
-   **数值积分与对比**: 通过数值积分对比 FFT 结果，并可视化展示 S(α) 的实部分布。
-   **主弧标注**: 在既约分数 a/q 处标注主弧 (Major Arcs)。
